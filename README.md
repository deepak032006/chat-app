📱 Real-Time Chat App

A full-featured real-time chat application built with the MERN stack and Socket.IO.
The app supports private messaging, typing indicators, online user tracking, and stores chat history securely.

🌐 Live Demo

🚀 Features

🔒 User Authentication with JWT

💬 Real-Time Messaging using Socket.IO

👥 Private & Group Chats

✍️ Typing Indicators

🟢 Online/Offline User Tracking

📜 Persistent Chat History with MongoDB

🛠️ Tech Stack

Frontend: React.js, CSS

Backend: Node.js, Express.js

Database: MongoDB

Real-Time: Socket.IO

Authentication: JWT

Deployment: Vercel

📂 Project Setup
1️⃣ Clone the repo
git clone https://github.com/deepak032006/chat-app.git
cd chat-app

2️⃣ Install dependencies
npm install

3️⃣ Add environment variables

Create a .env file in the root with the following:

MONGO_URI=your_mongo_database_url
JWT_SECRET=your_secret_key
PORT=5000

4️⃣ Start the app
npm run dev
